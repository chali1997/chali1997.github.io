How To Install IDEA2023 Community

· 2023-09-14 · # Other

1. 下载

官网下载：https://www.jetbrains.com/zh-cn/idea/download/download-thanks.html?platform=windows&code=IIC

这是 IntelliJ IDEA Community 版本，现阶段不需要 Ultimate 的额外功能，Community 已足够使用，没必要去折腾破解

2. 安装

下载完成后会得到这样一个安装包

双击安装，进入到安装界面

点击“是”

点击“Next”至下面界面

从上至下，从左到右，分别为创建桌面快捷方式，添加到系统变量，添加把文件夹作为项目打开的右键菜单，文件关联。自行按需选择，接着点击”Next“进入安装阶段

安装完成

3. 汉化

安装完成后初始为英文，切换至中文便于我们上手

第一次进入需要我们同意用户协议，勾选后点击”Continue“

进入到主界面，点击"Plugins"来到插件页面

搜索”Chinese“关键词后找到下图插件，点击Install安装汉化包，汉化完成

4. 新建项目

点击”新建项目“

进入到下图界面

如图所示，项目名称，项目存放位置，根据自己需求修改，强烈推荐修改项目位置方便记忆管理。

**特别注意：**JDK 设置会自动识别，如果识别不到可以点开JDK设置 --> 添加JDK --> JDK；如果没有安装 JDK，则可以JDK设置 --> 添加JDK-->下载JDK

添加示例代码会自动生成一份包含 Hello World 的 main 文件，自行判断是否勾选

最后点击创建项目完成创建

5. 创建 Java 文件

来到 IDEA 界面，在 src 文件夹右键选择Java类即可创建 Java 文件，一定要在 src 文件夹创建 Java 文件，以防产生额外的问题
