How to install FFmpeg on CentOS Stream 9

· 2023-09-14 · # Linux

  • 步骤一 First Step

    确保系统软件为最新

    sudo dnf clean all
sudo dnf update

  • 步骤二 Second Step

    启用 EPEL 和 RPMfusion 存储库

    运行以下命令以安装 EPEL 存储库

    sudo dnf install https://dl.fedoraproject.org/pub/epel/epel-release-latest-9.noarch.rpm
sudo dnf install https://dl.fedoraproject.org/pub/epel/epel-next-release-latest-9.noarch.rpm

    下一步运行一下命令以安装 RPM Fusion 存储库

    sudo dnf install --nogpgcheck https://mirrors.rpmfusion.org/free/el/rpmfusion-free-release-$(rpm -E %rhel).noarch.rpm
sudo dnf install https://mirrors.rpmfusion.org/nonfree/el/rpmfusion-nonfree-release-$(rpm -E %rhel).noarch.rpm

  • 步骤三 Third Step

    默认情况下，FFmpeg 在 CentOS Stream 9 基础存储库中可用，使用以下命令即可安装最新版本的 FFmpeg

    如果遇到nothing provides ladspa needed by rubberband-3.1.3-1.el9.x86_64 from epel报错

    是因为系统缺少 ladspa 软件包，通过执行dnf --enablerepo=crb install ladspa安装即可解决

    可在https://pkgs.org/查找软件包

    sudo dnf install ffmpeg

    对于开发版本dev

    sudo dnf install ffmpeg-free-devel

    确认安装并检查已安装 FFmpeg 构建版本

    ffmpeg -version

  • 后续使用请参考官方文档FFmpeg Documentation
